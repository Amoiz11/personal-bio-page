# Abdul Moiz - Bio Page

> 🌱 **This is my very first web development project!** Built entirely from scratch using only HTML and CSS as I started my journey into web development.

A clean, responsive personal bio/profile page built using **HTML5** and **CSS3**. This project showcases my personal and professional information including skills, education, interests, and a contact form.

## 🔗 Live Preview
https://amoiz11.github.io/personal-bio-page/


## ✨ Features
- Responsive hero header with profile image
- About Me section
- Interests & Skills list
- Education details in a styled table
- Favorite things table
- Social media & email contact links
- Interactive feedback form with styled inputs
- Fully responsive design (mobile-friendly)

## 🛠️ Built With
- **HTML5** – Semantic page structure
- **CSS3** – Custom styling, Flexbox, CSS variables, gradients, transitions


## 📚 My Learning Journey
As my **first project** after completing the basics of HTML, I built this page to practice everything I learned. Since this is just the beginning of my web development journey, this project focuses on fundamentals rather than advanced techniques. Through it, I practiced:
- Semantic HTML structure (`header`, `main`, `footer`)
- Working with tables, lists, and forms
- CSS styling fundamentals (colors, spacing, typography)
- CSS variables and reusable design tokens
- Responsive design with media queries
- Hover effects and transitions

More projects are on the way as I continue learning! 🚀
